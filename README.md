Generate a playlist of randomly selected tracks from newly released albums on [BandCamp](https://bandcamp.com/) by genre.

Built with Vue & Bootstrap 4

## Installation

* `npm install`
* `node app.js`

Head over to [http://localhost:5000](http://localhost:5000)
